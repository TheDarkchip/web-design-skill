# Mobile & responsive design

## Baseline
- Design for narrow viewports first; scale up.
- Avoid horizontal scroll; prefer stacking.

## Touch
- Targets are comfortably tappable; give breathing room between actions.
- Avoid tiny icon-only buttons without labels or tooltips.

## Layout patterns
- Use a single column with sections.
- Collapsible filters, sticky primary CTA when appropriate.

## Content prioritization
- Move secondary content below primary task area.
- Use progressive disclosure for advanced options.
