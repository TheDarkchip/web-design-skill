# Graspable Web Design Coach (Codex Skill)

This is a Codex Agent Skill. Install it by placing the folder in a Codex skills directory, for example:

- Repo-scoped: `.codex/skills/graspable-web-design-coach/`
- User-scoped: `~/.codex/skills/graspable-web-design-coach/`

Then restart Codex and invoke it via `$graspable-web-design-coach` (or let Codex auto-trigger it).

## Contents
- `SKILL.md` — skill metadata + instructions
- `references/` — concise guideline cards (heuristics, accessibility checks, layout rules)
- `assets/` — report templates + token starter CSS + sample HTML
- `scripts/` — optional deterministic audit (`ui_audit.py`)
